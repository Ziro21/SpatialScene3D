# Colab Notebooks
This directory contains Google Colab notebooks for GPU-dependent stages.
These notebooks are designed to run on Colab (free tier T4 or Pro A100).

## Notebooks
- `colab_pipeline.ipynb` — Full cloud pipeline: MASt3R-SLAM → gsplat → Grounded-SAM-2

## Workflow
1. Upload your video frames to Google Drive
2. Open the notebook in Colab
3. Follow the cells step by step
4. Download the outputs (COLMAP workspace, .ply, masks) to your Mac
